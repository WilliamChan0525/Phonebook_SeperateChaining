I created a phonebook with seperate chaining hash table. I know this might not be the best data structure to store it, but it just because I learned hash table recently so that I want to use it for practicing.

I used an unsorted singly linked list for it since it would be more efficient for inserting items, which takes O(1).

Time complexity:
  Searching: O(n)
  Inserting: O(1)
  Deleting: O(n)
